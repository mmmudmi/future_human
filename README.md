# AGI Awakening - The Journey Within

A thought-provoking interactive VR experience that explores ethical questions about AI and humanity's future. **This project raises awareness about the urgent need for responsible AI development and human-centered decision making.** Players make critical choices that determine which of four unique endings they receive based on their values and priorities.
